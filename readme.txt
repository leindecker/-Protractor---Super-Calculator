Protractor Demo Project

This is a demo project about Protractor and Jasmine framework.

Structure:
--- css
	-- report style.
--- framework
	-- Classes with sharing method's.
--- pages
    -- Page Objects pattern.
--- reports
    -- Jasmine Reports.
--- scenarios
    -- Test file folder.
	
conf.js: Protractor configuration file, is mandatory to run your testes using Protractor.

	
How to run:
'sh run_tests.sh' or 'protractor conf.js'
